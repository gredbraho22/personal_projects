# Trade Visualizer – Trading Journal & Performance Dashboard

Modern, clean trading performance visualizer built with React, TypeScript, Tailwind CSS, shadcn/ui and Recharts.

**Features**
- CSV import (compatible with Thinkorswim, Interactive Brokers, Tradovate, NinjaTrader, etc.)
- Key trading statistics: Total PnL, Win Rate, Expectancy, Profit Factor, R-multiples, drawdown
- Visualizations: Equity curve, monthly heatmap, trade distribution
- Risk & position sizing calculator
- Filters (symbol, strategy, session, weekday…)
- Dark/light mode + fully responsive

## Quick start

```bash
git clone https://github.com/YOUR_USERNAME/Trade-Visualizer.git
cd "Trade Visualizer"
npm install
npm run dev
